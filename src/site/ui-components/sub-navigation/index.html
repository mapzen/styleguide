Sub-navigation

The sub-navigation is used when there are multiple pages in a section. It can have up to three levels of information.

This component will take up 100% width of its parent container. To put it on the side of a page, place it inside a grid containing element, such as .col-sm-2 for a two-column wide navigation element. On mobile screens, this component should be the full width of the device.

  • Use the ARIA roles 'directory', 'navigation', or 'complementary' where appropriate
  • Use the .affix-* classes to force it to never scroll off the page
  • The .nav base class is from Bootstrap and some functionality is inherited from it
